# Media — Prompts de génération d'images

Images à générer pour le site vitrine de Coup de Main Étudiant.
Style cible : illustrations flat/vectorielles, couleurs vives (corail, turquoise, jaune), ton jeune et communautaire, québécois.

## hero.webp (1920×800)

```
Flat vector illustration, wide banner format.
A diverse group of cheerful young students helping in a suburban Quebec neighbourhood: one mowing a lawn, one carrying a moving box, one painting a fence.
Bright coral, turquoise, and warm yellow colour palette.
Friendly, energetic vibe. No text. Clean white background edges for overlay.
```

## services.webp (800×600)

```
Flat vector illustration, grid of small icons style.
Icons representing: lawn care, cleaning, light moving, furniture assembly, event help, general handiwork.
Bright coral, turquoise, and warm yellow colour palette on white background.
Simple, minimal, cheerful.
```

## comment-ca-fonctionne.webp (1200×400)

```
Flat vector illustration, horizontal 3-step infographic style.
Step 1: person filling a form on a phone. Step 2: handshake between student and client. Step 3: student happily working in a yard.
Connected by dotted arrows. No text, no numbers, no labels.
Bright coral, turquoise, yellow. White background.
```
